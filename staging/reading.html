Textbooks

This course utilizes many freely available textbook resources and lab instructions will reference these textbooks for reading assignments with the abbreviations below. Use the links below to access the textbook resources.

  • LCL: The Linux Command Line – William Schotts

  • PID: A Practical Introduction to Databases – Christopher Painter-Wakefield

  • TCS: How to Think Like a Computer Scientist: Interactive Edition

  • FWP: Fundamentals of Web Programming

  • OSM: Operating Systems and Middleware: Supporting Controlled Interaction

  • CODF: Computer Organization and Design Fundamentals – David Tarnoff

  • ICN: An Introduction to Computer Networks

  • TDS: How to Think like a Data Scientist
